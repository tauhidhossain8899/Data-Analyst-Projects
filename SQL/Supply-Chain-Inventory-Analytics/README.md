# Supply Chain & Inventory Performance Analytics Using SQL

## Project Overview

This project analyzes inventory performance, supplier efficiency, logistics operations, and revenue generation using SQL.

## Business Problem

Supply chain inefficiencies can lead to:
- Stock shortages
- Overstocking
- Increased shipping costs
- Supplier delays

The goal of this project is to identify operational improvement opportunities through SQL analysis.

## Dataset

Supply Chain Dataset (100 Records)
